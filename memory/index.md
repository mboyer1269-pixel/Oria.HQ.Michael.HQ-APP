# Memory Vault index

| Type | Directory | Contents |
|------|-----------|----------|
| decision | `decisions/` | Recorded strategic/operational decisions |
| note / sop | `notes/` | Operational notes and procedures |
| source / doc | `sources/` | Reference documents and external sources |
| agent | `agents/` | Agent operating profiles |
| venture | `ventures/` | Venture memory |
| action | `actions/` | Action proposals and executions |
| chainline | `chainlines/` | Declared source → note → decision → action → ledger → pr → next paths |

See `README.md` for the entry format and rules.
