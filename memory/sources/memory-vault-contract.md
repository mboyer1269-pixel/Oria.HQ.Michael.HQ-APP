# Memory Vault contract (docs/MEMORY_VAULT_CONTRACT.md)

Canonical contract for the workspace-bound Memory Vault: entry types, trust
levels (verified / proposed / draft), author rules, Joris read rules (verified
only, max 20, workspace-scoped), and the locked capabilities list (Supabase
persistence, vector search, automatic ingestion, cross-workspace reads).
